# STM32F401 Bare-Metal Embedded Drivers

This repository contains my embedded systems practice and driver development using the STM32F401RB microcontroller.  
All programs are written in Embedded C using register-level programming (bare-metal, no HAL).

---

## 🚀 Objectives

- Understand microcontroller architecture deeply
- Develop drivers from scratch without HAL libraries
- Practice embedded C concepts like bitwise operations, memory-mapped registers, and interrupts
- Build a strong foundation for embedded software engineering roles

---

## 🧠 Concepts Covered

- Register-level programming
- Bitwise operations
- Memory-mapped I/O
- Polling vs Interrupts
- Peripheral interfacing

---

## 📂 Modules Implemented

### 🔹 GPIO
- LED blinking
- Input button handling
- Bitwise register control

### 🔹 UART
- Serial communication (TX/RX)
- Baud rate configuration
- Data transmission using polling

### 🔹 ADC
- Analog to digital conversion
- Sensor input reading
- Continuous and single conversion modes

### 🔹 LCD (16x2)
- LCD interfacing (4-bit mode)
- Command and data handling
- Displaying strings and numbers

### 🔹 Interrupts
- External interrupts (EXTI)
- NVIC configuration
- Interrupt handling routines

### 🔹 Polling
- Peripheral status checking
- Blocking-based communication

---

## 🛠️ Tools & Environment

- Microcontroller: STM32F401RB (Nucleo Board)
- IDE: Keil uVision
- Language: Embedded C
- Debugging: ST-Link Debugger
